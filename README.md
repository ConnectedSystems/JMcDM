# JMcDM
A package for Multiple-criteria decisio  making techniques in Julia


## Implemented methods

### MCDM Tools

- Topsis
- Electre
- Dematel
- Moora
- Vikor
- AHP
  
### SCDM Tools

- minimax
- maximin
- minimin
- maximax
- Savage
- Hurwicz
- MLE
- Laplace
- Expected Regret

### Game

- Game solver for zero sum games
